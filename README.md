# SIRENCode2040Hackathon-
For the Feb 1st Code 2040 Hackathon!

## Quick Start (Web Demo)
1. Open a terminal at the repo root.
2. Run:
   - `cd starter`
   - `npm install`
   - `npx expo start --web --clear`
3. Open the URL printed in the terminal (usually `http://localhost:8081`).

If you see a blank page, try refreshing after granting browser camera/mic permissions.
